# Radial Basis Function (RBF)

## Description
Implementation of **Radial Basis Function** to predict sine function `sin(x)`

## Getting Started
### Dependencies
`keras`
\
`tensorflow`
\
`matplotlib`
\
`numpy`

### Executing

this implementation is on `ipynb` environment.I also offer you to run this on `google colab`
